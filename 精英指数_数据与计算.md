# 怎样用数据看「精英指数」？

在荷兰中学（HAVO/VWO）语境下，「精英」可以从**学历层次、学科难度、毕业质量**三个维度用数据刻画。下面给出可操作的指标定义、数据来源和计算方式。

---

## 一、「精英」可以对应哪些数据维度？

| 维度 | 含义 | 可用数据 |
|------|------|----------|
| **学历层次** | 学校是更偏大学预科（VWO）还是高职预科（HAVO） | VWO 考生数 / (HAVO + VWO)，现有 JSON 可算 |
| **学科难度/取向** | 偏理科（N&T、N&G）常与理工、医学等“高选拔”路径关联 | N&T、N&G 占比（或 N&T 单独），现有 JSON 可算 |
| **毕业质量** | 考试通过率、成绩水平 | 通过率 = 通过人数/考生数；DUO 同一 CSV 有 GESLAAGDEN，可算 |

三者结合，就能用数据定义一个「精英指数」。

---

## 二、数据从哪里来？

### 1. 现有数据（本目录 JSON）

- **profielen_per_examenjaar**：各校每年 HAVO/VWO 各 profiel 的**考生数**（examenkandidaten）。
- 可汇总得到：
  - **VWO 占比** = VWO 总人数 / (HAVO + VWO)
  - **理科占比** = (N&T + N&G + N&T/N&G) / 总人数（HAVO+VWO）
  - **N&T 占比** = N&T 人数 / 总人数（若想突出“最难”理科）

### 2. 同一份 DUO CSV 里还有「通过人数」

- 数据源：**Examenkandidaten en geslaagden**（我们已用同一文件做 profiel 分布）。
- 每行除 EXAMENKANDIDATEN（考生数）外，还有 **GESLAAGDEN**（通过人数）、**SLAAGPERC**（通过率%）等列，按学年、按 profiel。
- 做法：按学校（BRIN）汇总所有 HAVO+VWO 的考生数和通过数，再算  
  **通过率 = 总通过人数 / 总考生数**（可只取最近一年或多年平均）。

---

## 三、精英指数的几种算法（只考虑 HAVO+VWO）

### 方案 1：单一维度（最简单）

- **精英指数 = VWO 占比**  
  - 公式：`VWO / (HAVO + VWO)`，取值 0–1。  
  - 含义：越高越偏“大学预科”，常被视作学校学术层次的一个指标。

或：

- **精英指数 = N&T 占比**（或 N&T+N&G 占比）  
  - 含义：越高越偏“高难度理科”，可视为学科精英取向。

数据：**仅用现有 JSON 即可**，不需额外下载。

---

### 方案 2：双维度合成（推荐）

把「学历层次」和「学科取向」合成一个 0–1 的指数，例如：

- **精英指数 = 0.6 × VWO占比 + 0.4 × 理科占比**

或（若更看重“最难理科”）：

- **精英指数 = 0.5 × VWO占比 + 0.5 × N&T占比**

- 两个占比都按 0–1 计算（分母为该校 HAVO+VWO 总人数）。  
- 数据：**仅用现有 JSON** 即可。

---

### 方案 3：加入「毕业质量」的复合指数

在方案 2 基础上加入通过率（需从 DUO CSV 里算）：

- **通过率** = 该校 HAVO+VWO 总通过人数 / 总考生数（最近一年或多年平均）。
- 将通过率归一化到 0–1（例如除以 100，或按全省分布做 min-max）。
- 例如：  
  **精英指数 = 0.4×VWO占比 + 0.3×理科占比 + 0.3×通过率（归一化）**

这样「精英」同时包含：层次（VWO）、取向（理科）、质量（通过率）。  
数据：VWO/理科用现有 JSON；通过率需解析 DUO CSV 的 GESLAAGDEN 列并按校汇总。

---

## 四、实施时要注意什么？

1. **只算 HAVO+VWO**：VMBO 不参与，避免把不同层次混在一起。  
2. **组合 profiel**：N&T/N&G 建议全算进「理科」；E&M/C&M 算进「文科/社科」。  
3. **`<5` 处理**：与 profiel 分布一致，如按 2 或 3 计，或该条不参与计算；建议在图中注明。  
4. **年份**：单年（如 2023-2024）或多年平均均可；多年更稳，单年更反映近期。  
5. **Futuris**：若只做 HAVO/VWO，Futuris 无 HAVO/VWO 数据可标为「无数据」或单独说明。

---

## 五、小结：用数据看「精英指数」的三种方式

| 方案 | 公式（示意） | 数据来源 |
|------|--------------|----------|
| 1. 单一 | 精英指数 = VWO占比 或 N&T占比 | 现有 JSON |
| 2. 双维（推荐） | 精英指数 = 0.6×VWO占比 + 0.4×理科占比 | 现有 JSON |
| 3. 复合 | 精英指数 = 0.4×VWO + 0.3×理科 + 0.3×通过率 | JSON + DUO CSV（GESLAAGDEN） |

- 若**先快速看、且不解析通过率**：用 **方案 1 或 2**，只读现有四校 JSON 即可算出并排序/画图。  
- 若**要包含“毕业质量”**：在 DUO 的「Examenkandidaten en geslaagden」里按 BRIN 汇总 GESLAAGDEN 与 EXAMENKANDIDATEN，得到通过率后按 **方案 3** 合成精英指数。

需要的话，可以再写一个脚本：从 JSON（和可选 CSV）读入 → 按选定方案算精英指数 → 输出表格或用于二维图（例如 X=理科占比、Y=VWO占比，点大小或颜色=精英指数）。
